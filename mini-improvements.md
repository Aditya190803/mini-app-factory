# Mini App Factory: Future Improvements

This document tracks planned professional IDE features and UX enhancements to elevate the platform's capabilities.

## 1. Right-Click Context Menus
- [ ] Implement custom context menus for files and folders.
- [ ] Options: Rename, Delete, Duplicate, and "New File in this Folder".
- [ ] Improved accessibility for users who prefer mouse-driven workflows.

## 2. Asset Upload & Management
- [ ] Support for binary assets (PNG, JPG, GIF, WebP).
- [ ] Drag-and-drop assets directly into folders.
- [ ] Integration with the ZIP export system.
- [ ] Previewer for image files.

## 3. SEO & Metadata Dashboard
- [ ] Centralized view for all pages and their metadata.
- [ ] Manage `<title>`, `<meta description>`, and `<og:image>` at a glance.
- [ ] Warning system for pages missing SEO data.

## 4. Smart Navigation Syncing
- [ ] Automatic link updates: If `about.html` is renamed to `about-us.html`, update all references across the project.
- [ ] Link Auditor to flag broken `<a>` tags or broken `src` attributes.

## 5. Keyboard Shortcuts
- [ ] `Ctrl + S`: Force manual sync/save.
- [ ] `Ctrl + B`: Toggle Explorer sidebar.
- [ ] `Ctrl + P`: Quick open (fuzzy search for files).
- [ ] `Del`: Open delete confirmation for the active file.

## 6. Breadcrumbs
- [ ] Add a breadcrumb navigation bar at the top of the Code Editor.
- [ ] Example: `assets > css > components > button.css`.
- [ ] Clickable segments for quick navigation.

## 7. Bulk AI Actions
- [ ] Multi-file instructions: "Apply dark mode to all pages."
- [ ] Code Refactoring: "Move all inline styles to `globals.css`."
- [ ] Project-wide Polish: "Standardize button styling across all partials."

## 8. Favicon Support
- [ ] Simple UI to upload a favicon or select an emoji.
- [ ] Automated injection into the `<head>` of all assembled pages.
